# Efficient-and-High-Fidelity-Multimodal-Machine-Unlearning-for-Information-with-Deep-Memory
Official PyTorch implementation of EMMU, an efficient information-level multimodal machine unlearning framework. EMMU identifies a small set of crucial parameters associated with sensitive information and updates only those parameters, achieving efficient unlearning while maintaining high model fidelity.

# Code
The repository includes the core implementation of EMMU.
